📊 Amazon Sales Data Analysis Dashboard (Power BI)

📌 **Project Description**

This project presents an interactive Power BI dashboard built to analyze Amazon sales data and uncover meaningful business insights. The dashboard helps in understanding sales performance, customer behavior, and product trends through visually rich and dynamic reports.
The goal of this project is to demonstrate strong data analytics, data visualization, and business intelligence skills using Power BI.

🎯 **Objectives**

Analyze overall sales performance 

Identify top-selling products and categories 

Understand customer purchase behavior 

Track revenue and profit trends 

Provide data-driven insights for decision making 

📁 **Project files**

Amazon sales data analysis.pbix → Main Power BI dashboard file

📊 **Dashboard Features**

🔹 **Sales Overview**

Total sales and revenue summary

Time-based sales trends (monthly/yearly)

🔹 **Product Analysis**

Best-selling products

Category-wise performance

Contribution of each category to total sales

🔹 **Customer Insights**

Customer purchase patterns

Repeat vs new customers (if included)

🔹 **Regional Analysis**

Region-wise sales distribution

High-performing locations

🔹 **Profit Analysis**

Profit margins

Revenue vs profit comparison

🛠 **Tools & Technologies Used**

Power BI Desktop

Data Cleaning & Transformation (Power Query)

Data Visualization Techniques

Basic Data Modeling

🚀 **How to Run This Project**

Clone or download this repository

Open the .pbix file using Power BI Desktop

Interact with filters, slicers, and visuals to explore insights
